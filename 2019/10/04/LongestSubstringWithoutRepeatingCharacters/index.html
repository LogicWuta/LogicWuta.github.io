LongestSubstringWithoutRepeatingCharacters

题目

给定一个字符串，请你找出其中不含有重复字符的 最长子串 的长度。

示例 1:

输入: “abcabcbb”
输出: 3
解释: 因为无重复字符的最长子串是 “abc”，所以其长度为 3。
示例 2:

输入: “bbbbb”
输出: 1
解释: 因为无重复字符的最长子串是 “b”，所以其长度为 1。
示例 3:

输入: “pwwkew”
输出: 3
解释: 因为无重复字符的最长子串是 “wke”，所以其长度为 3。
请注意，你的答案必须是 子串 的长度，”pwke” 是一个子序列，不是子串。

链表

思路：将字符串中的字符循环存如链表中，每次存入时进行查重，将链表头部移动到重复元素的next节点，并将链表长度减去删去的部分，每次插入后的长度都要和变量max进行对比，大于max则修改max值为当前链表长度。

class node{
    public char val;
    public node next;

    public node(char c){
        this.val = c;
        this.next = null;
    }
}

class ListC{
    public node first;
    public node last;
    public int count;

    public ListC(node n){
        this.first = n;
        this.last = n;
        count = 1;
    }

    public void isReaptedAndPush(node n){
        node tmp = first;
        int c = 0;
        while (tmp != null){
            if (tmp.val == n.val){
                count = count - c;
                last.next = n;
                last = n;
                first = tmp.next;
                return;
            }
            c++;
            tmp = tmp.next;
        }
        last.next = n;
        last = n;
        count++;
        return;
    }
}
class Solution {
    public int lengthOfLongestSubstring(String s) {
        int l = s.length();

        if (l < 2){
            return l;
        }

        ListC list = new ListC(new node(s.charAt(0)));
        int max = list.count;
        for (int i = 1; i < l; i++){
            list.isReaptedAndPush(new node(s.charAt(i)));
            if (list.count > max){
                max = list.count;
            }
        }

        return max;



    }
}

还有更优的HashMap解法，将字符串的每个字符放入HashMap中作为Key，字符在字符串中的位置作为Value。定义Start，End map变量，初始值Key为第一个字符，Value为-1和0，每次字符放入HashMap中End值变为那个放入的字符。当将检验到重复Key时，将Start值更改为重复的Key值和对应的Value后再修改HashMap中的Value值。计算长度的方法为End的Value减Start的Value，最大长度计算与链表法相同，不多赘述。
